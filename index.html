Настройка подключения на iOS SSTP Connect

1. Создать новое подключение, нажав + в правом верхнем углу приложения и выбрав New Profile
2. В Полях Description и Server указать ваш домен вида XXXX.keenetic.link
3. Выключить TLS Server Verification, подтвердив в доп.окне
4. В разделе EAP Configuration включить Enable TLS 1.3
5. В Основном разделе Authentication ввести назначенные Username и Password
6. Сохранить настройки, нажав в правом верхнем углу Save, подтвердить создание VPN-конфигурации
7. В основном меню VPN Status включить подключение с именем вида XXXX.keenetic.link
